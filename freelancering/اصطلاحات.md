| اصطلاح                             | معنی                                                                                                                                           |
| ---------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| **Ongoing contract**               | یعنی پروژه یک‌بار مصرف نیست، بلکه به‌صورت مداوم ادامه دارد (مثلاً هر ماه چند پروژه مشابه می‌فرستند). یعنی اگر خوب کار کنی، مشتری ثابت می‌مانی. |
| **Part-time contract**             | یعنی پروژه به‌صورت نیمه‌وقت است؛ در هفته چند ساعت کار مشخص.                                                                                    |
| **Fixed-price**                    | مبلغ کل پروژه از قبل مشخص است (مثلاً £150 برای کل کار). اگر زودتر انجام دهی، سود بیشتری می‌کنی.                                                |
| **Hourly contract**                | پرداخت بر اساس ساعت کاری (مثلاً £25/hr یعنی هر ساعت ۲۵ پوند).                                                                                  |
| **Remote**                         | یعنی پروژه از راه دور است (نیازی به حضور فیزیکی نیست).                                                                                         |
| **Deliverables**                   | خروجی‌هایی که باید تحویل دهی (مثلاً فایل STEP، PDF، BOM).                                                                                      |
| **Proposal**                       | پیشنهاد همکاری که تو برای پروژه می‌فرستی (شامل قیمت، زمان، و نمونه‌کار).                                                                       |
| **NDA (Non-Disclosure Agreement)** | توافق‌نامه محرمانگی؛ یعنی نباید جزئیات پروژه را به کسی بگویی.                                                                                  |
| **IP assignment**                  | یعنی تمام حقوق مالکیت طرح برای کارفرماست، نه برای تو.                                                                                          |
| **Concept Design**                 | طراحی مفهومی اولیه بدون جزئیات نهایی. بیشتر برای پیشنهاد و ایده‌پردازی.                                                                        |
| **BOM (Bill of Materials)**        | فهرست اجزای موردنیاز یک محصول (مثل پیچ، بلبرینگ، قطعات، متریال).                                                                               |
